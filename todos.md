## 專案目標
1. 串接 newsapi.org 所提供的文字新聞或是 Youtube 提供的影音
2. 在瀏覽器上即可瀏覽內容
3. 在 RSS 閱讀器也可以瀏覽訂閱的主題

目前已經先把基底部署一份至：https://nou-python.maid.tw

## 使用技術
* Python 3.x 
* Flask 網站開發框架
* 網站放置在 Google Cloud Platform 的 Cloud Run 服務上執行
